# Yandex GPT Telegram Bot with Notifications

## Features
- Интеграция с YandexGPT-lite
- Система уведомлений
- Хранение данных в JSON
- Планировщик задач

## Setup
1. `npm install`
2. Create `.env` file
3. `npm start`

## Environment Variables
See `.env.example`
